Go RSS Reader

Tech stack: Go (Golang), Gorilla/mux, REST API, React JS, HTML, CSS, MySQL

An RSS Reader with support of user sessions using HTTP cookies. The internal crawler indefinitely checks for the latest RSS updates, the process uses multithreading/concurrency. Fetched XML data is parsed and stored in the SQL database.

 

Go File Server

Tech stack: Go (Golang), Gorilla/mux, REST API, JavaScript, React JS, Webpack, Babel, HTML, CSS

A simple file server that supports essential operations such as listing directories, downloading and uploading files, renaming and deleting files/directories.

 

Restaurant Review App

Tech stack: JavaScript, React JS, Node.js, NGINX, PostgreSQL, AWS EC2, k6, Loader.io, New Relic

The app displays a short description of a chosen restaurant along with its location and various types of scores. The backend handles 1200 RPS while having 10,000,000 entries in the database. The app is horizontally scaled and deployed to multiple AWS EC2 hosts.

 

Vacation Rental

Tech stack: JavaScript, React JS, Express.js, Node.js, Webpack, Babel, MySQL, AWS S3, AWS EC2

A suggestion module for a vacation rental app that displays 12 more homes. Each picture is clickable and redirects the user to the main page of the clicked house. The app is responsive to resizing.

 

Sudoku

Tech stack: JavaScript, Express.js, Node.js, React JS, Webpack, Babel, MongoDB

Classic sudoku game. A user can start a new game or load previously saved (into the database) game. The application checks for validity of user input and whether game is over.

 

Python File Server

Tech stack: Python, Flask

A simple file server operates via REST API. Supports list, upload and delete operations.
